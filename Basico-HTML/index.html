História dos computadores

Os computadores começaram como máquinas enormes nos anos 1940, chamadas ENIAC. Eles eram usados principalmente para fazer cálculos complicados. Com o tempo, eles ficaram menores e mais inteligentes, até que surgiram os computadores pessoais, que podíamos ter em casa. Hoje, os computadores são super rápidos e fazem de tudo, desde enviar mensagens até criar filmes e jogos.

O que são Clients

Os Clients são os 'usuários' se serviços ou recursos fornecidos por um servidor. É como quando você acessa um site - você é o Cliente pedindo coisas para o Servidor.

O que são Servers

Os servers são os computadores que armazenam e fornecem dados e serviços para outros dispositivos.

Estrutura básica do HTML

A estrutura básica do HTML é dividida em duas partes: a parte da cabeça (head) e a parte do corpo (body) do código. Na parte do head, são definidas todas as especificações do site, como o idioma, o título do site, codificação de caracteres e outras informações que não são visíveis na página. Enquanto isso, o body contém todo o conteúdo visual, como os títulos, parágrafos, listas, botões, entre outros.

Principais tags

As principais tags para criação de um projeto na web com html são:

  • tag de título do mais importante para o menos importante:

    • <h1>

    • <h2>

    • <h3>

    • <h4>

    • <h5>
    • <h6>
  • tag de parágrafo: <p>
  • tag de grifar: <mark>
  • tag de texto com tamanho menor:<small>
  • tag de texto em negrito:<b>
  • tag de texto em Itálico:<em>
  • tag de texto sublinhado:<ins>
  • tag de lista ordenada, onde cada item é marcado com um número: <ol>
  • tag de lista não ordenada, onde cada item é marcado com um ponto: <ul>
  • tag de item de lista, usado dentro de <ol> ou <ul>
  • tag para criar links para outras páginas ou recursos na web <a>
  • tag de linha horizontal para separar o conteúdo: <hr>
  • tag que define texto que deve ser exibido como subscrito, ou seja, abaixo da linha de base norma: <sup>
  • tag que define texto que deve ser exibido como sobrescrito,ou seja, acima da linha de base normal: <sub>
  • tag que define um bloco de citação:
    <blockquote>
  • tag que deleta um texto: <del>

links

links são elementos fundamentais para conectar diferentes páginas da web entre si ou ancorar uma parte específica de uma página.

  • Link para outra página: <a> href="http://www.exemplo.com" > Visite Exemplo </a>
  • Abrir link da página na outra página: <a href="/http://www.exemplo.com" target=”_blank” > Visite Exemplo </a>
  • link para um item (usando o id) na mesma página: <a href="#secao1">Ir para Seção 1</a>
